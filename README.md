# Консольное приложение для взаимодействия с данными в файле JSON

## Запуск проекта

Приложение выведет текущее содержимое файла data.json


```
go run main.go
```

## Добавление и обновление данных

Значения сохраняются в формате строки (string)


```
go run main.go set <key> <value>
```

## Удаление данных

Данные удаляются по ключу


```
go run main.go delete <key>
```
